New Features and Functionality in Version 22.0

The HTML version of the VA FileMan Getting Started Manual contains the documentation updates based on the new features and functionality introduced with the release of VA FileMan V. 22.0 and any subsequent patches. This Web page contains the following update information:

  • Updates following the release of VA FileMan V. 22.0

  • Updates introduced with the release of VA FileMan V. 22.0

  • Previous Updates (pre-Version 22.0 and following the original release of VA FileMan Version 21.0 documentation in December of 1994)


Updates Following the Release of VA FileMan V. 22.0

The following table lists all significant patch updates, in reverse chronological order.

Date Description of Update Patch/Version Number
05/01/00 Updated the "DATE/TIME" topic to allow users to abbreviate minutes using the apostrophe ('). Patch DI*22*36
11/15/99 Added documentation about Exiting with the ^^. Patch DI*22*4


Updates/Changes Introduced with VA FileMan V. 22.0

The following table lists all significant updates introduced with VA FileMan V. 22.0, by category.

Category Description of Update
Inquire Option Added the "DISPLAY AUDIT TRAIL?" Prompt.
VA FileMan Prompts Added the Key Fields topic.
Added documentation regarding "Exiting with the Up-Arrow (^^)".

Updated the Adding a Duplicate Record topic with regard to Keys.
