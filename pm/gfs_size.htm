Global File Structure

22.2Overridable Maximum Global Node Sizes

As of 2016, at the time that this manual is updated, Cache supports 32KB Global Nodes, and GT.M can support up to 1MB global nodes. Fileman 22.0 could only support 255 long global nodes in compliance with the last approved M standard. In version 22.2, this limitation can be overridden by setting ^DD("STRING_LIMIT"). What this does is that option MODIFY FILE ATTRIBTES and any Replace ... With now will honor a programmer's or user's wish to enter long strings.
