VA FileMan Error Codes

Error 304

DESCRIPTION:

The problem with this IEN string is that it lacks the final ','. This is a common mistake for beginners.

TEXT:

The IENS '|IENS|' lacks a final comma.

PARAMETERS:

'IENS' means IENS.
