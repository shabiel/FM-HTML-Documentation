Classic VA FileMan API

EN^DIKZ: Compile

EN^DIKZ recompiles a file's cross-references by setting the input variables without user intervention.

Input Variables
X

The routine name.
Y

The file number of the file for which you want the cross-references recompiled.

DMAX

The maximum size the compiled routines should reach. Consider using the $$ROUSIZE^DILF function to set this variable.
