Database Server (DBS) API

CLEAN^DILF: Array and Variable Cleanup 

Reference Type Category ICR #
Supported Database Server (DBS) 2054
Description

The CLEAN^DILF API KILLs the standard message arrays and variables that are produced by VA FileMan.

Format
CLEAN^DILF
Input Parameters

None.

Output

The CLEAN^DILF API KILLs the following arrays:

  • ^TMP("DIERR",$J)
  • ^TMP("DIHELP",$J)
  • ^TMP("DIMSG",$J)

The CLEAN^DILF API KILLs the following variables:

  • DIERR
  • DIHELP
  • DIMSG
  • DUOUT
  • DIRUT
  • DIROUT
  • DTOUT
Error Codes Returned

None.
