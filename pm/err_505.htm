VA FileMan Error Codes

Error 505

DESCRIPTION:

The field name passed is ambiguous. It cannot be determined to which field in the file it refers.

TEXT:

There is more than one field named '|1|' in File #|FILE|.

PARAMETERS:

'1' means Field name.

'FILE' means File #.
