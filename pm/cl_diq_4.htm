Classic VA FileMan API

DT^DIQ: Display

This call converts the date in Y exactly like D^DIQ. Unlike D^DIQ, however, it also writes the date after it has been converted.

Input
Y

(Required) Contains the internal date to be converted. If this has five or six decimal places, seconds are automatically returned.

Output Variable
Y

External form of the date or date/time value, e.g., JAN 01, 1998.
