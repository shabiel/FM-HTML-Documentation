Internationalized Data Dictionary elements

File names, field labels, set values, and help messages can be entered into the ^DD schema for any of the languages listed in File .85.


DO SPANISH^DIALOGZ will allow the user to enter prompts that will be seen by users whose DUZ("LANG") number is 3.

D LANG^DIALOGZ(3) does the same thing.

D LANG^DIALOGZ(n) will do the same thing for the language in the LANGUAGE File  (File .85) whose number is 'n'.
