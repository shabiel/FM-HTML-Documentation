Attribute Dictionary: ^DD(Filenumber

  • Other Field Definition Nodes

  • Reading the Attribute Dictionary: An Example

The attribute dictionary describes the data fields that a file contains within the global ^DD (the data dictionary). Each attribute dictionary is stored descendent from a positive- valued, first-level subscript of this global. Each attribute dictionary, in itself, is also in the form of a file, and hence consists of entries, cross-references, descriptor, and a reference to the data dictionary of the attribute of attributes (^DD(0)).
