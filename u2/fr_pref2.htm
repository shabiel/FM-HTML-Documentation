Preface

The "VA FileMan Advanced User Manual" is designed to provide you, the VistA user, with "how to" information on the functions of VA FileMan, VistA's database management system. The features of VA FileMan are used throughout in most VistA applications.

This manual shows how to use features of VA FileMan that are likely to be used by ADP Application Coordinators (ADPACs), Information Resource Management (IRM) staff, and other technical users.

The following major features of VA FileMan are introduced along with a description on how to use them:

  • Advanced edit techniques

  • Importing and exporting data

  • Computed expressions

  • VA FileMan functions

  • Defining files and fields

  • Data security

  • Archiving data

  • System management for VA FileMan
