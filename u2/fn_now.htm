VA FileMan Functions

NOW

Format: NOW
Parameters: (none)
Use: This argumentless function returns the current date and time.
Example: NOW => AUG 23,1991 11:23

NOTE: For tips on displaying date-valued elements such as this function in computed expressions (e.g., printing) see the "Data Types in Computed Expressions" section.
