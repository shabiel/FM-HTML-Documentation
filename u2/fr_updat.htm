New Features and Functionality in Version 22.0

The HTML version of the "VA FileMan Advanced User Manual" contains the documentation updates based on the new features and functionality introduced with the release of VA FileMan Version 22.0 and any subsequent patches. This Web page contains the following update information:

  • Updates following the release of VA FileMan V. 22.0

  • Updates introduced with the release of VA FileMan V. 22.0

  • Previous Updates (pre-Version 22.0 and following the original release of VA FileMan Version 21.0 documentation in December of 1994)

Updates/Changes Introduced with VA FileMan V. 22.0

The following table lists all significant updates introduced with VA FileMan V. 22.0, by category.

Category Description of Update
List File Attributes Updated the List File Attributes option to include two new listing format options:
  • INDEXES AND CROSS-REFERENCES ONLY

  • KEYS ONLY
Creating Files and Fields Added a new Screen Mode interface when using the Modify File Attributes option.
Updated the overall "Creating Fields" topic to include revised Scrolling Mode examples with links to the new Screen Mode examples.
Updated the "Examples of File and Field Creation" topic to include samples using Screen Mode when creating or modifying fields with the Modify File Attributes option.
File Utilities Added a new Screen Mode interface when using the Edit File option. Also, included a new example in Screen Mode.
Updated the Template Edit option to now allow you to edit the contents of a template. Also, included new examples in Screen Mode.
Renamed the Cross-Reference a Field option to Cross-Reference a Field or File option and updated it with regards to cross-references:
  • Added the new New-Style Cross-references.

  • Added the word "Traditional" to the original cross-references topics in order to distinguish them from the New-Style cross-references.
Updated the Mandatory/Required Field Check option with regard to Keys.
Added the Key Definition option.
Updated the "Identifier" topic with regard to Keys.
Extract Tool Updated the "Important Items to Note" topic with regard to Keys.
Archiving Updated the "Find Archived Entries" topic with regard to Keys.
Updated the Archiving "Move Archived Data to Permanent Storage" topic with regard to Keys.
Updated the "Create Filegram Archiving Template" topic with regard to Keys.
