Previous Updates

The HTML version of the "VA FileMan Advanced User Manual" contains documentation updates since the original release of VA FileMan Version 21.0 documentation in December of 1994 (including new functionality introduced by patches). This page lists all significant updates, in reverse chronological order. For specific updates based on the release of VA FileMan Version 22.0 and any subsequent patches, see the "New Features and Functionality in Version 22.0" topic in this manual.

Date Description of Update Patch/Version Number
06/19/98 Changed title from "User 2" to "Advanced User" manual. Version 21.0
06/02/98 Additional revisions made to the format and layout of the manual to improve usability:
  • Added Chapter links on every topic page so that if page is reached on its own (outside of frameset) you can still get to the Chapter (as well as Contents) top pages.

  • Used cascading style sheet to apply a uniform style to all pages, for users of Internet Explorer 4x and greater, and Netscape 4x and greater.

  • The style sheets use the designed-for-Web-readability Verdana font (download it!). If not present on the system, falls back to Arial, Helvetica or Sans Serif.
 Version 21.0
04/16/98 Revised the layout of the manual as follows:
  • Went from three to two frames.

  • Removed all javascript that 'auto-synched' the frames.

  • Added Contents link in header so that if page is reached on its own (outside of frameset) you can still get to the Contents.
 Version 21.0

10/02/96 Created a new "tabs" style top frame for the manual set. Eliminated the bottom frame also - less frames (three vs. four before) should mean quicker loading. Version 21.0
09/25/96 All files now get their "last updated" date directly from the file's time/date stamp, via the javascript document.lastModified property. Also, the "framesynching" javascript function will no longer cause an error if a page is loaded on its own, outside of the context of the frameset for this manual.

All cross-manual links (between API and U1 manual, for example) now use "../" syntax to change directories (instead of a full "http:..." path), so the cross-manual links should work, even if the manuals are downloaded to and installed on a PC.

Version 21.0
(inception) What was the "VA FileMan User Manual" is now, in the HTML version, split into two manuals:
  • "Getting Started Manual" -- how to use basic VA FileMan features (for all VA FileMan users).

  • "Advanced User Manual" -- how to use more advanced VA FileMan features (for ADPACs, developers, and IRM staff)
This manual, the "Advanced User Manual," is now more tightly focused on topics of interest to experienced VA FileMan users. It is split into four sections, covering:
  1. Tools and Techniques

  2. File Management

  3. Security

  4. Archiving
Some of the information in the "Advanced User Manual" has been reorganized and revised since the release of the original V. 21.0 "VA FileMan User Manual" in December of 1994. 		Version 21.0
