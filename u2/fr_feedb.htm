Sending Feedback

Send us feedback at G.FMTEAM@FORUM.VA.GOV (click on this address if your Web browser is set up to send email.)

For you to be able to send us feedback from your Web browser, it must be set up correctly. Here are some tips in case you have trouble getting your mail to transmit for the first time using a Web browser:

  • Netscape Navigator

    1. Outgoing Mail (SMTP) Server setting: From the Netscape menu, select Options, then Mail and News Preferences. From the Servers submenu, in the Outgoing Mail (SMTP) Server field, enter the TCP/IP address of the hospital computer system from which you access your local e-mail (your IRM office can provide this information).

    2. Set an address for Incoming Mail: Although the value in the Incoming Mail (POP) Server field won't be used, you need to set a "dummy" value there anyway. Enter "Mail" for this field.

    3. Identity: Select the Identity submenu. Enter your name in the (surprise!) Your name field. Likewise, enter your Forum email address in the Your E-mail field (if you do not have a Forum address try using your local email address).

  • Microsoft Internet Explorer for Windows 95

    1. Use Microsoft Internet Mail instead of Microsoft Exchange. In Options, under the Programs Tab, for the Mail field, choose Internet Mail. You can only choose this if your PC has Microsoft Internet Mail installed on it. This is available as a download from Microsoft.

    2. Outgoing Mail (SMTP) Server setting: From Options, under the Servers tab, set the Outgoing Mail (SMTP) field to the TCP/IP address of the hospital computer system from which you access your local e-mail (your IRM office can provide this information).

    3. Set an address for Incoming Mail: Although the value in the Incoming Mail (POP3) field won't be used, you need to set a "dummy" value there anyway. Enter the same value here as for the Outgoing Mail (SMTP) field.

    4. Identity: Enter your name in the Name field, and enter your email Forum mail address goes in the Email Address field (if you do not have a Forum address try using your local e-mail address).

  • Other Browsers

    1. Since the underlying method of sending mail from a Web browser is the same no matter what the browser, try to see what configuration options in your browser correspond to the features listed above that must be set in Netscape Navigator and Microsoft Internet Explorer.
