Extract Tool

Print Selected Entries Option (3 of 9)

To display the list of entries you have selected, use the Print Selected Entries option. This option uses the standard VA FileMan interface for printing (see the "Print: How to Print Reports from Files" chapter of the "VA FileMan Getting Started Manual"). The example that follows depicts the type of dialogue you will encounter when printing a list of entries to be extracted: 

Select EXTRACT OPTION: PRINT SELECTED ENTRIES
Select EXTRACT ACTIVITY: 3   CHANGE 08-30-92  EDITED
 SELECTOR:SMITH,J  EXTRACT

Enter a regular Print Template name or fields you wish to see 
printed on this report of records to be extracted.

FIRST PRINT FIELD: [ZZTEST TEMPLATE

The output looks like:

CHANGE EXTRACT ACTIVITY                AUG 30, 1992  11:09   PAGE 1
NO.                  VERSION           PROGRAMMER
-----------------------------------------------------------------
101                  17.10             DOE,JANE
102                  17.32             DOE,JOHN
103                  17.35             DOE,JOHN
330                  17.09             DOE,JULIA
