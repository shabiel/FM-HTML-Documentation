System Management

^DINIT

Before any user program is used, VA FileMan's DINIT routine must be run on your system to initialize VA FileMan (see the "VA FileMan Installation Guide"). The DINIT routine needs to be run only once, although it is nondestructive if run again.

 22.2DINIT can be run without the presence of the VA Kernel software; if FileMan is being installed on a system with the rest of Kernel, see also the Installation Guide for information about installing VA FileMan.
